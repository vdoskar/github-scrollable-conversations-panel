# GitHub Sticky PR Panel

Jednoduché Chrome rozšíření, které na stránce GitHub Pull Requestu udrží pravýpanel konverzace viditelný při posouvání stránky. Pokud je panel vyšší než okno, lze jeho obsah posouvat samostatně.

## Instalace

1. Otevřete v Chromu `chrome://extensions`.
2. Zapněte **Developer mode**.
3. Klikněte na **Load unpacked**.
4. Vyberte tuto složku.
5. Otevřete nebo obnovte stránku Pull Requestu na `github.com`.

Rozšíření nepotřebuje žádná dodatečná oprávnění, build ani instalaci závislostí.
